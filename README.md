# mjml-browsersync
MJML email templating using BrowserSync livereload

Usage:
```
git clone git@github.com:iamandrewluca/mjml-browsersync.git your-folder
cd your-folder
npm install
```

Start with live reload:
```
npm run start
```

Just compile:
```
npm run build
```

Compiled files will go into `dist` folder
